Objetos e Arrays - Day 03

Possibilidade de adicionar um conjunto de valores a uma variável;

O array deve ser escrito entre colchetes, separando os valores por vírgulas, veja:

let numeros = [1, 3, 5, 8, 11]

let informacoes = ["Bruno", 12, "Idade", 26]

Podemos acessar um array pelo seu índice, por exemplo: numeros[1]

Lembrando que o primeiro índice sempre é 0, ou seja, para acessar o primeiro elemento: numeros[0]
