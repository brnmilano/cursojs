Destructuring com array - Day 05

Podemos definir variáveis com os valores de um array utilizando o destructuring;

let nomes =[ 'João', 'Maria', 'José', 'Antônio', 'Miguel', 'Fernanda' ]; let [nomeA, nomeB, nomeC, nomeD, nomeE, nomeF] = nomes; console.log(nomeA); console.log(nomeB); console.log(nomeC);
